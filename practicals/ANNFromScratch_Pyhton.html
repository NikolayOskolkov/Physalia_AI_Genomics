XOR Non-Linear Problem ¶

XOR was an example of a problem that could not be solved with a simple Perceptron model. The reson was that an ANN without hidden layers could solve only linear problems while XOR was a simplest example of a non-linear problem.

ANN without a hidden layer is capable of making a linear desicion boundary. From the figure above we can see that the XOR problem can not be solved with drawing a single linear decision boundary, i.e. crosses and circles can only be separated with a non-linear surface, like the read curve in the figure, or by two linear surfaces. Therefore an ANN with a hidden layer is required to solve the XOR problem.

Solving XOR in Keras ¶

We can easily implement the solution of the XOR problem with an ANN in Keras. Here we are going to use 1 hidden layer with 2 hidden neurons in addition to the input and output layers. Activation functions will be Sigmoid for both hidden and output layers. We will use the Mean Square Error (MSE) loss function and Stochastic Gradient Descent (SGD) with the Learnig Rate (lr) equal to 0.1, i.e. quite a large learning rate. Since we have a binary otcome, it would probably make more sense to use the Binary Cross-Entropy cost function, however here we will try to make the ANN as simple as possible, hence we choose the MSE cost function.

In [6]:
import numpy as np
from tensorflow.keras.optimizers import SGD
from tensorflow.keras.models import Sequential
from tensorflow.keras.layers import Dense

X = np.array([[0,0],[0,1],[1,0],[1,1]])
y = np.array([[0],[1],[1],[0]])

model = Sequential()
model.add(Dense(2, input_dim = 2, activation = 'sigmoid'))
model.add(Dense(1, activation = 'sigmoid'))

model.compile(loss = 'mse', optimizer = SGD(lr = 0.1))
model.fit(X, y, epochs = 100000, verbose = 0)

model.predict(X)

We can see that the ANN with one hidden layer and two hidden neurons can easily solve the XOR problem and get correct assignments to the 4 observations (2 crosses and 2 circles). However, please note the large number of training epoches (epoches = 100 000) that was needed to get the model converged to the true labels. This is because we are using the simpest possible architecture of ANN, i.e. with MSE cost function and only 1 hidden layer and only 2 hidden neurons. In addition we are using SGD and sigmoid activation functions. Those hyperparameters are not the smartest choice, and their better choise can dramatically decrease the training time.

Deriving and Coding ANN from Scratch ¶

Now we will implement an ANN with all the elements derived and programmed from scratch. Below we present the ANN architecture to be used. It includes the input layer taking 2 vectors x1 = (0 0 1 1) and x2 = (0 1 0 1), and 1 hidden layer with 2 hidden neurons h1 and h2. The output y gets a prediction from the ANN and compares it against the true labels d = (0 1 1 0).

Both hidden and the output nodes have the sigmoid activation function that has a very simple derivatve that can be expressed through the function itself:

$$ \phi(s) = \frac{1}{1+e^{\displaystyle -s}}; \,\,\, \phi\prime(s)=\phi(s)\left(1-\phi(s)\right) $$

Then we can easily write down the input for each of the hidden neurons:

$$ h_1 = \phi(x_1 w_{11} + x_2 w_{21}) \\ h_2 = \phi(x_1 w_{12} + x_2 w_{22}) $$

The above two equations can be formalizes in a linear algebra matrix form as follows:

$$ h_j = \phi\left(\sum_{i=1}^2{x_i w_{ij}}\right); \,\,\, x_i = (x_1 \,\,\, x_2) = \begin{pmatrix} 0 & 0\\ 0 & 1\\ 1 & 0\\ 1 & 1 \end{pmatrix}; \,\,\, h_j = (h_1 \,\,\, h_2); \,\,\, w_{ij}= \begin{pmatrix} w_{11} & w_{12}\\ w_{21} & w_{22}\\ \end{pmatrix} $$

Please note that since $h_j$ is a vector-row with dimensions (1x2), and $x_1$ and $x_2$ are vector-columns with dimensions (4x1), the resulting output of the hidden layer, h, will be a matrix with dimensions (4x2). The output of the ANN can be expressed as a sigmoid of the linear combination of the two hidden neurons:

$$ y = \phi\left(\sum_{j=1}^2{h_j w_{hj}}\right); \,\,\, w_{hj} = \begin{pmatrix} w_{h1}\\ w_{h1} \end{pmatrix} $$

Here $w_{hj}$ is a vector-column with dimensions (2x1). Since $h_j$ was a matrix (4x2) and $w_{hj}$ was a vector (2x1), the product, y, will be a vector-column with dimensions (4x1). This output (prediction, y) will be compared against the true labels d, and the error will be given as quadratic deviation of the prediction, y, from the truw labels, d.

$$ E = \frac{1}{2}\left(d-y\right)^2 = \frac{1}{2}\left(d-\phi\left(\sum_{j=1}^2{h_j w_{hj}}\right)\right)^2 $$

The hidden weights $w_{hj}$ and input weights $w_{ij}$ are updated via gradient descent from the computed error in such a way that first the hidden weights are updated from the error, and then the input weights are updated, implying that the updating occurs backwards, hence it is called backpropagation.

$$ w_{hj} = w_{hj} - \mu\frac{\partial E(w_{hj})}{\partial w_{hj}};\\ \frac{\partial E(w_{hj})}{\partial w_{hj}} = \left[d-\phi\left(\sum_{j=1}^2{h_j w_{hj}}\right)\right]* \phi\left(\sum_{j=1}^2{h_j w_{hj}}\right)*\left(1-\phi\left(\sum_{j=1}^2{h_j w_{hj}}\right)\right)*h_j\\ w_{ij} = w_{ij} - \mu\frac{\partial E(w_{ij})}{\partial w_{ij}};\\ \frac{\partial E(w_{ij})}{\partial w_{ij}} = \frac{\partial E(w_{hj})}{\partial w_{hj}}*w_{hj}* \phi\left(\sum_{i=1}^2{x_i w_{ij}}\right)\left(1-\phi\left(\sum_{i=1}^2{x_i w_{ij}}\right)\right)*x_i $$

Now, once we have gotten the expressions for the error gradinets, everything is ready for programming the forward and backward propagations. We start with defining the vectors $x_1$ and $x_2$, combining them into a matrix (4x2) and defining the vector d, which is the true labels of the 4 XOR samples that we are going to learn. Please note that d is a vector-column with dimensions (4x1).

def sigmoid(x):
    return 1 / (1 + np.exp(-x))
def grad_sigmoid(x):
    return sigmoid(x) * (1 - sigmoid(x))

Now using the expression for the gradient of the mean square error with respect to the hidden weights above, we can use the error = d - y to update the hidden weights. Here $y*(1-y)$ is simply the derivative $y\prime$ of the prediction, y, at the output node. Please note the the matrix product of $dE_dwh$ and $h_j$ will be done later on the step when we update the hidden weights via gradient descent.

Next, the error gradient with respect to the hidden weights, dEdwh, is included into the expression for the error gradient with respect to the input weights $w{ij}$. Again, here $h(1-h)$ is the derivative, $h\prime$, of the output of the hidden node, $h$. Since dE_dwh has dimensions (4x1) and $w_h$ has dimensions (2x1), we have to transpose $w_h$ in oder to compute its matrix product with dE_dwh. As a result, the dimensions of the dE_dw matrix is going to be (4x2), i.e. the same as the h-matrix. One more thing to note is that the matrix multiplication is done via np.dot() numpy function while element-wise multiplication is simply done via the asterics () operator.

Finally, after we computed the error gradients with respect to the hidden and input weights, we are going to update the hidden weights $w_h$ and input weights $w_{ij}$ via gradient descent. On the fly, we will multiply the input $x_j$ and output of the hidden node $h_j$ by the gradients dE_dw and dE_dwh, respectively following matrix multiplication rules. Here we select the lerning rate mu = 0.1 for faster convergence of the gradient descent procedure, this will work for this simple task, although normally for real-world applications one should use smaller lerning rates.

Great, seems like we can follow and code both forward and backward propagation steps. Let us now merge all the codes above together into a single piece. Repeating the update rule iteratively, we obtain a simple loop:

In [8]:
import numpy as np

x1 = np.array([0, 0, 1, 1])
x2 = np.array([0, 1, 0, 1])
d = np.array([[0],[1],[1],[0]])
x = np.array([[i, j] for i,j in zip(x1, x2)])

def sigmoid(x):
    return 1 / (1 + np.exp(-x))
def grad_sigmoid(x):
    return sigmoid(x) * (1 - sigmoid(x))

mu = 0.1
epochs = 100000
w = np.array([[0.1, 0.5], [0.2, 0.9]])
wh = np.array([[0.9], [0.9]])

E = list()
for i in range(epochs):
    
    #Forward propagation
    h = sigmoid(np.dot(x, w))
    y = sigmoid(np.dot(h, wh))
    
    #Backward propagation
    E.append(np.array((d - y)**2).sum())
    dE_dwh = (d - y) * y * (1 - y)
    wh = wh + mu * np.dot(h.T, dE_dwh)
    
    dE_dw = np.dot(dE_dwh, wh.T) * h * (1 - h)
    w = w + mu * np.dot(x.T, dE_dw)
